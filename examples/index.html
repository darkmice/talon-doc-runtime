这是什么

TDR 让 Agent（Claude 等）输出**短小、语义明确**的 HTML 标签 — <d> / <src> / <branch> / <evidence> / <myth> 等 — 浏览器加载 runtime 后，DOM 被原地增强为带主题、折叠、代码高亮、引用跳转和受控事件的可交付文档。

设计目标只有一个：让 Agent 用尽可能少的 token，输出结构稳定、视觉成熟、可复用的文档制品。

两分钟集成

npm install @talon-ui/doc-runtime
 
<html data-archetype="business-document">
  <body>
    <main class="tdr-doc">
      <h1>Your structured doc</h1>
      <d s="bad" v="P0" t="Bug title">...</d>
    </main>
    <script src="https://unpkg.com/@talon-ui/doc-runtime/dist/talon-doc-runtime.iife.js"></script>
  </body>
</html>

完整的标签清单、属性、组件选型指南见 README 与 SKILL.md。
